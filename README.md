# GRATIS DIGITAL ASSESSMENT

This serves as my submission for the Gratis Digital junior backend developer position. It is a basic NodeJs REST API which fulfils the very basic requirements of a Blog App.

[x] A user can create a blog post
[x] A user can view a blog post
[x] A user can delete a blog post
[x] A user can update the details of a blog post
[x] A user can add a comment to a post
[x] A user can view a comment on a post
[x] A user can delete a comment
[x] A user can update the details of a comment

# This project was built using Nodejs

### Libraries used

[x] Expressjs: Used to create the server and write the APIs
[x] Express-validator: Used to validate http request body
[x] Mocha: Used to test the API response ensuring it met all business requirements
[x] Chai: Javascript assertion library used alongside Mocha for testing
